Football Champions

How To Play This Game



Football Champions is a test about winners of various football tournaments. The goal is to test if you know the winners for 4 different tournaments. We have the World Cup, the African Nations Cup (AFCON), the Champions Leage and the Ballon D'or award.


To get the best of this an interested user is required to follow the follwing steps.


  • Input a user name and click on submit.

  • The game then starts with questions for the World Cup tournament.

  • For each tournament we have possible answers below where the user selects one and submits the answer.

  • For each answer submitted you immediately get a score increment and correction with supporting image.

  • Once questions for a particular tournament are finished you will be notified before continuing to the next.

  • The last set of questions are for the Ballon D'or award. After which the user will get final score.
